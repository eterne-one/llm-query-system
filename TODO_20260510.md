축하합니다! 이제 **번역(KO→EN) → LLM 추론(Gemma) → 한글 답변 유도**까지 이어지는 핵심 파이프라인이 로컬 환경에서 완벽하게 구동되는군요.

설계 문서의 흐름상, 이제 이 대화 데이터들을 휘발시키지 않고 안전하게 보관할 **데이터베이스(DB) 기반**을 다질 차례입니다.

---

## 넥스트 스텝: 데이터베이스 모델링 (`src/app/db/models.py`)

단순히 메모리에 저장하던 세션 이력을 실제 DB 테이블로 옮겨야 합니다. 설계 문서 요구사항에 맞춰 3가지 핵심 테이블을 정의하겠습니다.

### 1. 설계할 테이블 구조

* **Users (사용자)**: 로그인 기능을 위한 계정 정보 저장.
* **ChatHistory (대화 이력)**: 사용자의 질문(한글/번역문)과 모델의 답변을 매핑하여 저장.
* **ErrorLogs (에러 로그)**: 번역 실패나 LLM 호출 에러 발생 시 추적을 위한 기록.

### 2. 기술 스택

* **SQLAlchemy**: 파이썬의 대표적인 ORM으로, 객체 지향적으로 DB를 다룹니다.
* **SQLite**: 로컬 개발 환경이므로 별도의 서버 설치 없이 파일 형태로 가볍게 시작합니다.

## 코드 구현: `src/app/db/models.py`

이 코드를 해당 경로에 작성해 주세요.

```python
from sqlalchemy import Column, Integer, String, Text, DateTime, ForeignKey
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.sql import func
from sqlalchemy.orm import relationship

Base = declarative_base()

class User(Base):
    """사용자 정보를 담는 테이블"""
    __tablename__ = "users"

    id = Column(Integer, primary_key=True, index=True)
    username = Column(String(50), unique=True, nullable=False)
    password = Column(String(255), nullable=False) # 실제 운영 시 해시 암호화 저장
    created_at = Column(DateTime(timezone=True), server_default=func.now())

class ChatHistory(Base):
    """질문 및 답변 이력을 저장하는 테이블"""
    __tablename__ = "chat_history"

    id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey("users.id"))
    session_id = Column(String(100), index=True) # 세션별 묶음을 위함
    
    query_ko = Column(Text, nullable=False)      # 사용자 한글 입력
    query_en = Column(Text)                      # 번역된 영문 쿼리
    response_ko = Column(Text)                   # LLM의 최종 답변
    
    created_at = Column(DateTime(timezone=True), server_default=func.now())

    # 관계 설정
    user = relationship("User")

class ErrorLog(Base):
    """시스템 장애 로그 저장 테이블"""
    __tablename__ = "error_logs"

    id = Column(Integer, primary_key=True, index=True)
    error_type = Column(String(50))              # 번역오류, LLM오류 등
    error_message = Column(Text)                 # 에러 상세 내용
    created_at = Column(DateTime(timezone=True), server_default=func.now())

```
